КЗ-план: Дорожка 1 (ширина 1.2 м)

Заказ (по ширинам):
- 1.5 м: 3×3.8 м (38-15-8), 1×2.9 м (29-15-8) — суммарно 14.3 м
- 1.2 м: 2×6.3 м (63-12-8), 2×3.8 м (38-12-8) — суммарно 20.2 м
- 1.0 м: 1×6.3 м (63-10-8), 2×5.3 м (53-10-8), 1×3.8 м (38-10-8), 1×2.8 м (28-10-8) — суммарно 23.5 м

Правила резки:
- Плита 1.5 м → 1.2 м + 0.3 м (продольный рез)
- Ленту 0.3 м при необходимости режем до 0.2 м для комплектовки плит 1.0 м

Последовательность укладки (ширина 1.2 м):
1) Плиты 1.2 м — укладываются без реза (как есть).
2) Плиты 1.5 м — использовать как 1.2 м; дополнительно выдаются ленты 0.3 м для стыков.
3) Плиты 1.0 м — комплектуются лентами 0.2 м до ширины 1.2 м.

Разрезы (продольные по ширине):
- 1.5 → 1.2 + 0.3: 4 реза; получаем ленты 0.3×(3.8, 3.8, 3.8, 2.9)
- 0.3 → 0.2: 4 реза; получаем ленты 0.2×(3.8, 3.8, 3.8, 2.9)
- Донор 1.2×6.3: распустить на 4×0.3 (3 реза), затем 0.3→0.2 (4 реза) → ленты 0.2×(6.3, 6.3, 6.3, 6.3)
- Итого продольных резов: 15

Подрезы по длине:
- 0.2×2.9 → 0.2×2.8 (подрез 0.1)
- 0.2×6.3 → 0.2×5.3 ×2 шт (подрез 1.0 ×2)
- Итого подрезов по длине: 3

Комплектовка плит 1.0 м лентами 0.2 м:
- 3.8 м → +0.2×3.8 (из 1.5)
- 2.8 м → +0.2×2.8 (из 2.9; подрез 0.1)
- 5.3 м → +0.2×5.3 (из донора 6.3; подрез 1.0)
- 5.3 м → +0.2×5.3 (из донора 6.3; подрез 1.0)
- 6.3 м → +0.2×6.3 (из донора 6.3)

Остатки и отход:
- Неприменённые ленты 0.2: 1×6.3 м; 2×3.8 м; 2×1.0 м. Сумма: 0.2×15.9 м.
- Технологический отход от 0.3→0.2: ширина 0.1 м × длина (14.3 + 25.2) = 39.5 пог. м (≈ 3.95 м²).

Примечания:
- Все плиты одного армирования (8). При необходимости можно перераспределить донорские ленты 0.2 под другие длины.
- По запросу можно выпустить XLSX по шаблону с листами «Укладка», «Разрезы», «Остатки».
